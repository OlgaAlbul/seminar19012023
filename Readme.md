## Журнал изменений
Для просмотра журнала изменений используется команда *git log*. Для этого  в терминале с папкой-репозиторем пишем *git log*. В показанном журнале будут:
* Хэш(номер) коммита
* Дата и время коммита
* Автор коммита
* Текст сообщения к коммиту
